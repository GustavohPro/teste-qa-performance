📊 O QUE CADA SCRIPT FAZ:
═══════════════════════════════════════════════════════════════════════════════

📈 LOAD TEST (load-test.js)
   Objetivo: Simular carga constante de 250 usuários
   Tempo: 5 minutos
   Estágios:
   • 0-1 min:  0 → 50 VUs (aquecimento)
   • 1-3 min: 50 → 250 VUs (ramp-up)
   • 3-4 min: 250 VUs (sustain - carga máxima constante)
   • 4-5 min: 250 → 0 VUs (ramp-down)
   
   Critério de Aceita ção:
   ✓ 250 RPS (requisições por segundo)
   ✓ 90º percentil < 2 segundos
   ✓ Taxa de erro < 10%

📍 SPIKE TEST (spike-test.js)
   Objetivo: Simular pico abrupto de carga (2x a carga normal)
   Tempo: 3,5 minutos
   Estágios:
   • 0-1 min:    0 → 100 VUs (aquecimento leve)
   • 1-1.5 min: 100 → 500 VUs (spike abrupto!)
   • 1.5-2.5 min: 500 VUs (sustain no pico)
   • 2.5-3 min: 500 → 100 VUs (ramp-down rápido)
   • 3-3.5 min: 100 → 0 VUs (final)
   
   Teste de resiliência e picos inesperados
